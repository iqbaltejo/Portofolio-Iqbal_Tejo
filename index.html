Digital Transformation & IoT Implementation

Mengintegrasikan sistem digital dan perangkat IoT untuk meningkatkan efisiensi operasional, transparansi data, dan akurasi pengambilan keputusan di lingkungan industri. Setiap proyek dirancang dengan pendekatan terukur agar memberikan dampak nyata terhadap kecepatan, akurasi, dan kontrol operasional.

Aplikasi Surat Jalan

Sebelumnya, proses pengumpulan dan verifikasi surat jalan dilakukan secara manual dan bisa memakan waktu 1–3 hari (H+1 hingga H+3) setelah aktivitas pengiriman. Melalui pengembangan aplikasi surat jalan berbasis web, proses kini dapat dilakukan secara real-time, bahkan pada area tanpa jaringan—data otomatis tersinkronisasi saat petugas kembali ke kantor pada pergantian shift.

Selain itu, pada aktivitas ekspor dan impor, sistem ini menghubungkan dua lokasi pengawasan berbeda — Terbanggi Besar (pabrik) dan Waylunik (pelabuhan) — sehingga setiap aktivitas loading dan unloading kini dapat dipantau langsung tanpa bergantung pada laporan manual.
Hasil & Dampak :
  • Waktu pelaporan surat jalan berkurang hingga 90% (dari H+3 menjadi real-time).
  • Akurasi data aktivitas logistik meningkat hingga 95%.
  • Koordinasi antar lokasi operasional menjadi lebih cepat dan transparan.
Ringkas :
  • Aplikasi web untuk surat jalan tangki & ekspor-impor.
  • Pemantauan aktivitas loading/unloading real-time antar lokasi.
  • Efisiensi proses administrasi meningkat signifikan.

Sensor Adukan Agitator

Sistem ini dikembangkan untuk memantau proses pengadukan larutan pupuk pada unit tangki supply agar sesuai dengan standar kualitas yang dibutuhkan tanaman nanas. Sebelumnya, tidak ada data terukur mengenai aktivitas agitator, sehingga ketika hasil panen menurun, penyebabnya sulit diidentifikasi. Selain itu, saya juga membangun dashboard interaktif untuk sensor agitator, yang membantu manajemen dalam mengambil keputusan berbasis data.

Dengan pemasangan sensor agitator dan dashboard monitoring berbasis web, kini aktivitas pengadukan dapat dipantau dan direkam secara otomatis, memastikan konsistensi dan kualitas distribusi larutan ke area perkebunan.
Hasil & Dampak :
  • Pencatatan aktivitas agitator meningkat dari 0% menjadi 100% terdata otomatis.
  • Kualitas pengadukan larutan meningkat dan lebih konsisten antar unit.
  • Memberikan dasar data objektif untuk evaluasi produksi dan perawatan alat.
Ringkas :
  • Instalasi sensor agitator dengan dashboard web.
  • Monitoring kualitas pengadukan larutan pupuk secara real-time.
  • Menjamin konsistensi nutrisi tanaman dan mengurangi kesalahan operasional.

Fuel Sensor

Mengimplementasikan sensor bahan bakar (fuel sensor) pada armada operasional untuk memantau konsumsi dan distribusi BBM secara real-time. Data dikirim langsung ke platform web monitoring, memungkinkan analisis efisiensi berdasarkan jenis aktivitas lapangan.

Sistem ini juga memiliki fitur deteksi anomali yang dapat memberikan peringatan dini terhadap potensi pencurian bahan bakar.
Hasil & Dampak :
  • Visibilitas konsumsi bahan bakar meningkat hingga 100% real-time tracking.
  • Efisiensi penggunaan BBM meningkat sekitar 20–25% berkat analisis aktivitas.
  • Kasus kehilangan bahan bakar menurun secara signifikan setelah implementasi sistem.
Ringkas :
  • Integrasi fuel sensor dengan web monitoring.
  • Analisis efisiensi BBM berdasarkan aktivitas operasional.
  • Deteksi dini pencurian bahan bakar secara otomatis.

Rancang Bangun Peta Virtual

Pada masa pandemi COVID-19, mahasiswa baru tidak dapat mengikuti kegiatan orientasi dan pengenalan kampus secara langsung. Untuk menjawab tantangan tersebut, saya mengembangkan Aplikasi Virtual 3D Gedung Dekanat FISIP Universitas Lampung, yang memungkinkan calon mahasiswa menjelajahi lingkungan kampus secara interaktif.

Desain dimulai dari sketsa 2D, dikembangkan menjadi model 3D detail, dan akhirnya diintegrasikan ke dalam aplikasi virtual berbasis web. Melalui aplikasi ini, pengguna dapat mengenal area kampus, jurusan, dan fasilitas secara realistis tanpa harus datang langsung ke lokasi.
Hasil & Dampak :
  • Menjadi media pengenalan kampus virtual pertama di FISIP Unila.
  • Mempermudah calon mahasiswa mengenal jurusan dan lingkungan kampus selama pandemi.
  • Mendukung proses adaptasi digital di lingkungan pendidikan tinggi.
Ringkas :
  • Virtual 3D Gedung Dekanat FISIP Unila.
  • Solusi interaktif pengenalan kampus saat pandemi.
  • Menggabungkan desain 2D, 3D, dan aplikasi virtual berbasis web.

IT Infrastructure & System Reliability

Fokus pada pengembangan infrastruktur jaringan yang tangguh, efisien, dan mampu mendukung kebutuhan operasional digital secara berkelanjutan.

Dalam bidang infrastruktur, saya berfokus pada pembangunan jaringan yang stabil, andal, dan efisien untuk mendukung konektivitas lintas area. Saya merancang dan mengimplementasikan sistem jaringan berbasis mesh network dengan memanfaatkan perangkat CPE220 sebagai penangkap sinyal utama dan TL-MR3420 untuk redistribusi koneksi ke seluruh area kerja.

Pendekatan ini tidak hanya memperluas jangkauan sinyal, tetapi juga menciptakan sistem konektivitas berlapis (redundant) yang tetap berfungsi meskipun salah satu node mengalami gangguan.
Hasil & Dampak :
  • Cakupan jaringan meningkat hingga +60% dari area awal.
  • Koneksi menjadi lebih stabil dengan tingkat downtime berkurang hingga 40%.
  • Memastikan komunikasi data dan perangkat IoT berjalan tanpa gangguan pada area operasional.
Ringkas :
  • Implementasi mesh network dengan perangkat CPE220 & TL-MR3420.
  • Meningkatkan jangkauan dan kestabilan jaringan.
  • Mendukung sistem operasional dan IoT dengan konektivitas optimal.
