Digital Transformation & IoT Implementation

Aplikasi surat jalan berbasis web; Dashboard sensor BBM real-time; Visualisasi data IoT agitator

Saya mengembangkan berbagai solusi digital untuk mendukung efisiensi operasional, mulai dari aplikasi surat jalan berbasis web yang mempermudah administrasi, hingga sistem monitoring sensor BBM yang memberikan data konsumsi bahan bakar secara real-time. Selain itu, saya juga membangun dashboard interaktif untuk sensor agitator, yang membantu manajemen dalam mengambil keputusan berbasis data.

2D & 3D Design

Sketsa 2D control room; Modeling 3D dengan SketchUp; Aplikasi virtual 3D interaktif

Kreativitas saya dituangkan dalam desain ruang kerja, dimulai dari sketsa control room dalam bentuk 2D, kemudian diwujudkan menjadi model 3D menggunakan SketchUp. Dari sana, saya mengembangkan aplikasi virtual 3D yang memungkinkan pengguna berinteraksi langsung dengan desain tersebut secara imersif.

IT Infrastructure & System Reliability

Implementasi mesh network; CPE220 untuk penangkapan jaringan; TL-MR3420 untuk distribusi koneksi

Dalam bidang infrastruktur, saya berfokus pada pembangunan jaringan yang stabil dan dapat diandalkan. Saya merancang mesh network menggunakan perangkat CPE220 untuk menangkap sinyal jaringan dan TL-MR3420 untuk mendistribusikannya kembali, sehingga menghasilkan konektivitas yang lebih optimal dan efisien.

Pamflet
